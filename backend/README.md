# Retro - Backend

The Backend is a simple Express Server utilizing Websockets.

## Scripts

| Name         | Description                                      |
| ------------ | ------------------------------------------------ |
| start        | starts the server                                |
| format       | formats the whole backend project using Prettier |
| format:check | checks if formatting is violated                 |

## Dependencies

- Express
- Lodash
- Puppeteer
- SocketIO
